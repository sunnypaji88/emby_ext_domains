Welcome to **emby_ext_domains**! This lightweight Go application expands the features of your Emby media server. ## üåê Accessing the API
After starting the application, it will be accessible locally. You can interact with the API through your web browser or any tool that can make HTTP requests (like Postman).

### API Endpoint
The main endpoint to retrieve server domain names is:
```
http://localhost:YOUR_PORT/domains
```
Replace `YOUR_PORT` with the port number specified in the application output.
